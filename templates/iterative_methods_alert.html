Guía para Métodos Iterativos

Sigue estas pautas al ingresar datos para los métodos iterativos Gauss-Seidel, Jacobi, y SOR:

1. Matriz A
  • Debe ser una matriz cuadrada. Esto significa que el número de filas y columnas debe ser igual.
  • Los elementos deben ingresarse como filas separadas por punto y coma (;) y números separados por comas (,).
  • Ejemplo: Para la matriz: 
                    4 -1  0
                -1  4 -1
                 0 -1  4
    Ingresa: 4,-1,0; -1,4,-1; 0,-1,4
2. Vector B
  • Debe ser un vector columna con el mismo número de elementos que filas en la matriz A.
  • Los valores deben ingresarse separados por comas (,).
  • Ejemplo: Para el vector: 
    [15, 10, 10]
    Ingresa: 15,10,10
3. Vector Inicial (x0)
  • Debe tener el mismo tamaño que el vector B.
  • Se usa como punto de partida para los cálculos iterativos.
  • Los valores deben ingresarse separados por comas (,).
  • Ejemplo: 0,0,0 para inicializar todos los valores en 0.
4. Tolerancia (Tol) y Número Máximo de Iteraciones (Niter)
  • Tolerancia: Un número positivo pequeño (e.g., 0.001) que define la precisión del resultado.
  • Iteraciones: Un entero positivo que define el límite de iteraciones para evitar bucles infinitos.
5. Tipo de Error
  • Error Absoluto: Se calcula como la diferencia entre iteraciones consecutivas.
  • Error Relativo: Se calcula como la diferencia relativa entre iteraciones consecutivas.
Ejemplo Completo
  • Matriz A: 4,-1,0; -1,4,-1; 0,-1,4
  • Vector B: 15,10,10
  • Vector Inicial: 0,0,0
  • Tolerancia: 0.001
  • Iteraciones: 100
