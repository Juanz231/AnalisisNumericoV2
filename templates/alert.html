Cómo Ingresar Funciones

Para garantizar un cálculo correcto, sigue estas pautas al ingresar funciones matemáticas:

  • Exponencial: Usa np.exp(x) para ex
  • Logaritmo Natural: Usa np.log(x) para ln(x)
  • Potencias: Usa x**n para xn (e.g., x**2 para x²)
  • Funciones Trigonométricas: Usa np.sin(x), np.cos(x), y np.tan(x) para seno, coseno y tangente
  • Raíz Cuadrada: Usa np.sqrt(x) para √x
  • Valor Absoluto: Usa np.abs(x) para |x|
  • Constantes:
    • np.pi para π
    • np.e para e

Ejemplos de Entradas:

  • Para f(x) = e-x, ingresa: np.exp(-x)
  • Para f(x) = ln(x²), ingresa: np.log(x**2)
